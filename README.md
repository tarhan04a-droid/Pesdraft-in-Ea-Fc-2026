# PES Draft: Ea Fc 26 Simulation 🎲

An interactive, tactical EA FC 26 draft simulator that allows two players to draft squads, perform strategic trades, and optimize team formations based on real-world player datasets.

## 🚀 Key Features
- **Tactical AI:** Integrated "Formation Guarantee" algorithm that dynamically adjusts squad selection based on 20+ professional EA FC tactical formations.
- **Fair Draft Algorithm:** Uses high-performance data manipulation via `pandas` to ensure balanced and competitive squads.
- **Blind Trade Room:** A unique, interactive GUI mechanism allowing players to sacrifice "bait" players to steal specific targets from their opponent.
- **Data Sanitization:** Built-in security layer that automatically filters out structural dataset errors (e.g., unrealistic OVR values) to ensure simulation stability.

## 🛠 Tech Stack
- **Language:** Python
- **GUI:** Tkinter
- **Data Analysis:** Pandas
- **Imaging:** PIL (Pillow) & Requests

## ⚠️ Important Note
This project is a **standalone simulation engine**. It does not automatically fetch data from the web or modify in-game files. 

### Prerequisites
- Python 3.x installed.
- A local `.csv` file containing EA FC player data (with columns: `short_name`, `overall`, `player_positions`, `player_face_url`).
